# Чек-лист бизнес-критериев приёмки QNACF

## Критерии приёмки
- `criteria_001.md` - Пользователь может запустить приложение одной командой
- `criteria_002.md` - Пользователь видит вопросы и может выбирать ответы
- `criteria_003.md` - Система сохраняет ответы и создаёт новые вопросы
- `criteria_004.md` - Приложение работает в Docker контейнере
- `criteria_005.md` - Интерфейс обновляется при изменении файлов
- `criteria_006.md` - Система генерирует итоговые чек-листы
